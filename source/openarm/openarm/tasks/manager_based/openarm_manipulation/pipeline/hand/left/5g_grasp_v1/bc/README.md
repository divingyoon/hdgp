# 5g_grasp_v1 BC (t2 + Demo)

This folder provides a practical starting point for demo-based grasp BC.

## Files

- `retarget_allegro_to_tesollo.py`
  - Converts external hand demo sequence into Tesollo joint targets and action vectors.
- `collect_grasp_observations.py`
  - Replays action sequence in `5g_grasp_left-v1` and records the actual policy observations.
- `retarget_config_example.json`
  - Limit / offset / scale / mapping template used by the converter.
- `train_bc_mlp.py`
  - Trains an MLP behavior cloning regressor from `(observations, actions)`.

## Output Dataset Format (`.npz`)

Generated by `retarget_allegro_to_tesollo.py`:

- `actions`: shape `[T, action_dim]` (default action_dim=32)
- `tesollo_joints`: shape `[T, 20]` (`lj_dg_1_1..5_4`)
- `synergy`: shape `[T]` (`left_hand_action` scalar)
- `thumb_action`: shape `[T, 8]` (`left_thumb_action`)
- `meta_json`: JSON string metadata

For BC training, add `observations` (same action-aligned time steps) to the NPZ.
If `observations` is missing, `train_bc_mlp.py` can fallback to `tesollo_joints`.

## Typical Workflow

0. Build object bank (DexYCB + DexGraspNet)
1. Convert external demo
2. Build/align observations from your grasp rollout pipeline (or use `collect_grasp_observations.py`)
3. Run BC pretrain
4. Use BC weights as initialization for PPO fine-tuning

## Build Object Bank

```bash
python build_object_bank_from_demo.py \
  --dex-ycb-dir /home/user/rl_ws/demo/dex-ycb \
  --dexgraspnet-dir /home/user/rl_ws/demo/DexGraspNet \
  --out-dir ../assets \
  --max-dex-ycb 8 \
  --max-dexgraspnet 8
```

This generates:
- `../assets/object_bank.json`
- `../assets/object_bank.yaml`
- `../assets/generated_urdf/*.urdf`

The `5g_grasp_left-v1*` tasks load this bank to spawn different objects per episode.

## Demo Workflow

1. Convert external demo
2. Build/align observations from your grasp rollout pipeline (or use `collect_grasp_observations.py`)
3. Run BC pretrain
4. Use BC weights as initialization for PPO fine-tuning

## Example

```bash
python retarget_allegro_to_tesollo.py \
  --input /path/to/allegro_seq.npy \
  --config retarget_config_example.json \
  --output /tmp/demo_tesollo_grasp.npz

python collect_grasp_observations.py \
  --task 5g_grasp_left-v1 \
  --input-actions /tmp/demo_tesollo_grasp.npz \
  --output /tmp/demo_tesollo_grasp_with_obs.npz \
  --headless

python train_bc_mlp.py \
  --dataset /tmp/demo_tesollo_grasp_with_obs.npz \
  --out /tmp/bc_actor.pt
```
